# FlowLedger Design System

## Brand Identity
- **Name:** FlowLedger
- **Tagline:** Your Money Has a Memory Problem. We Fixed It.
- **Tone:** Surgical precision. No hype. Every word earns its place.
- **Vibe:** Palantir meets Stripe meets a scalpel

## Color Palette
- **Background:** #050508 (near-black with slight blue tint)
- **Surface:** #0d0d14 (elevated surface)
- **Glass:** rgba(255,255,255,0.04) with border rgba(255,255,255,0.08)
- **Accent Cyan:** #00f5c4 (electric mint — primary CTA, highlights)
- **Accent Blue:** #0070f3 (secondary accent)
- **Accent Purple:** #7928ca (depth/gradients)
- **Text Primary:** #ffffff
- **Text Secondary:** #a0a0b4
- **Text Muted:** #4a4a65
- **Danger/Kill:** #ff3b30 (kill switch red)
- **Grid Lines:** rgba(0,245,196,0.06)

## Typography
- **Heading:** Inter, weight 700-900, tracking -0.03em
- **Body:** Inter, weight 400, line-height 1.7
- **Mono/Data:** JetBrains Mono for numbers/code elements
- **Hero headline:** 72-96px, ultra-bold, tight letter-spacing
- **Section heading:** 48-56px
- **Body:** 17-18px

## Design Language
- **3D CSS transforms** — cards with perspective, tilt on hover
- **Particle/grid backgrounds** — animated canvas particles
- **Glassmorphism cards** — frosted glass with luminous borders
- **Gradient text** — cyan to blue on hero words
- **Glowing elements** — box-shadow glow on CTAs and icons
- **Scanline/grid overlays** — subtle tech grid aesthetic
- **Smooth scroll sections** — full viewport height sections
- **Micro-animations** — entrance animations, counter animations

## Key Sections
1. **Nav** — minimal, dark, sticky with blur
2. **Hero** — full viewport, particle bg, massive headline, animated 3D card mockup
3. **Problem** — visceral stats, bleeding money animation  
4. **Solution** — The Stitching Engine explained visually
5. **How It Works** — 3-step flow with animated connectors
6. **Features** — glassmorphism feature cards
7. **Roadmap** — Phase 1/2/3 timeline
8. **Waitlist CTA** — full-width with glow
9. **Footer**

## Component Style
- Border radius: 12-16px on cards
- All interactive elements: cursor-pointer with 200ms transitions
- Hover: slight upward lift (translateY -4px) + glow intensification
- Section padding: 120px vertical
